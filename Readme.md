## Examples

```bash
git clone ...
cd react-mobx-vm
npm install
npm run gojs:open -- -i routerV3/browserHistory.js 
```

## Tests
### Units
```bash
npm test
```

### Integration
```bash
npm run test:cy:run
# or 
npm run test:cy:open
```
